# 📱 RuangAman App

**RuangAman** adalah aplikasi berbasis Android yang dirancang sebagai platform konsultasi, edukasi, dan pelaporan terkait bullying. Aplikasi ini bertujuan memberikan ruang aman bagi pengguna untuk memahami, mencegah, dan menindaklanjuti kasus bullying secara lebih terstruktur.

---

## 🎯 Tujuan Aplikasi

- Memberikan edukasi tentang bullying dan pencegahannya
- Menyediakan wadah konsultasi sederhana
- Memfasilitasi pelaporan kasus bullying
- Meningkatkan awareness terhadap pentingnya kesehatan mental

---

## 🛠️ Teknologi yang Digunakan

- **Bahasa Pemrograman:** Kotlin
- **Platform:** Android
- **UI Component:** RecyclerView
- **Form Validation:** Input validation (tidak boleh kosong)

---

## ✨ Fitur Utama

### 1. 🔐 Registrasi (Basic)
- Pengguna dapat melakukan registrasi akun
- Validasi input:
    - Semua field wajib diisi
    - Tidak boleh kosong (empty/null validation)
- ⚠️ *Catatan:*
    - Belum terdapat sistem login
    - Data belum terintegrasi dengan backend/database

---

### 2. 🏠 Home (Edukasi Bullying)
- Menampilkan konten edukasi dalam bentuk artikel/blog
- Materi mencakup:
    - Pengertian bullying
    - Jenis-jenis bullying
    - Cara pencegahan
    - Dampak bullying
- Menggunakan **RecyclerView** untuk menampilkan list konten secara dinamis

---

### 3. 📝 Lapor Bullying
- Fitur untuk melaporkan kasus bullying
- Pengguna mengisi data seperti:
    - Nama (opsional/anonim jika dikembangkan)
    - Deskripsi kejadian
    - Lokasi / kronologi
- Validasi:
    - Tidak boleh ada field kosong
